CSC DCAF includes the following CSC projects which are kept under version control in a CSC GitHub repository:

  • DCAF - The DCAF project sources which contain all the automation modules to build the AutoDeployNode and deploy data center cloud infrastructures.
  • Hanlon - Hanlon is an advanced provisioning application which can deploy both bare-metal and virtual systems.
  • Slimer - The CSC Slimer project, a fork of abrezhnev/slimer, is used to deploy the Red Hat OpenStack Platform with high availability on base RHEL installations. For more information see the csc/slimer wiki documentation. When used with DCAF, the result is a bare metal installation of the RHEL OS and Red Hat OpenStack Platform with high availability.
  • Ansible-ScaleIO - The CSC Ansible-ScaleIO project, a fork of sperreault/ansible-scaleio, is used to install, configure and manage ScaleIO. For more information see the csc/ansible-scaleio wiki documentation. When used with DCAF, this project adds EMC ScaleIO storage capabilities to the Red Hat OpenStack Platform.
